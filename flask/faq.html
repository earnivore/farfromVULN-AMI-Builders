Frequently Asked Questions

How do I get a OpenVPN profile to connect to my AWS Virtual Private Cloud?

To download a OpenVPN profile, navigate to http://my_server_ip:7894/my_ovpn_file where my_ovpn_file is a random file name that you can choose!

How secure is farfromVULN?

Because farfromVULN opens up a webserver to distribute VPN profiles, it receives traffic from internet users that you may not have invited. Report security concerns to the project's GitHub page.
